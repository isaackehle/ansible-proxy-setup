# Ansible: proxy-setup

Configure a system to use a Proxy Server

Available on Ansible Galaxy: [pgkehle.proxy-setup](https://galaxy.ansible.com/pgkehle/proxy-setup)


# Examples

```YAML

  - hosts: all
  
    roles:
      - pgkehle.proxy-setup
```
